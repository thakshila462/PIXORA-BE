# Pixora Backend - API Server

##  Project Description
This is the backend of Pixora Full Stack Web Application built using Node.js, Express.js, TypeScript, and MongoDB.  
It provides RESTful APIs for authentication, user management, AI services, and service requests.

---

##  Tech Stack
- Node.js
- Express.js
- TypeScript
- MongoDB + Mongoose
- JWT Authentication
- bcryptjs
- CORS

--
##  Features
- User Authentication (Register/Login)
- JWT Token-based Security
- Password Encryption using bcryptjs
- Role-based Authorization (Admin/User)
- CRUD APIs for users, packages, and service requests
- AI API integration support (Gemini/OpenAI)
- Secure RESTful API architecture
